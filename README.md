# homebrew-tap

Homebrew tap for [Nimble](https://github.com/surdy/nimble) — a keyboard-driven command palette for your desktop.

## Install

```sh
brew tap surdy/tap
brew install --cask nimble
```

## Update

```sh
brew update
brew upgrade --cask nimble
```

> **Note:** This tap is updated automatically on each release via the release CI in the main repository.
