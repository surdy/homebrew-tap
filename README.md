# homebrew-tap

Homebrew tap for [Context Actions](https://github.com/surdy/contexts-launcher) — a cross-platform command-driven desktop launcher.

## Install

```sh
brew tap surdy/tap
brew install context-actions
```

## Update

```sh
brew update
brew upgrade context-actions
```

> **Note:** This tap is updated automatically on each release via the release CI in the main repository.
